# Alert

Bootstrap alerts provide contextual feedback messages for typical user actions with a handful of available and flexible alert messages.

## Overview

Alerts are available for any length of text, as well as an optional dismiss button. They can be styled with one of the eight contextual variants (primary, secondary, success, danger, warning, info, light, dark).
